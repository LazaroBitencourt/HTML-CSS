Utilizando as primeiras tags aprendidas

Tags em HTML

Em HTML, as tags são elementos fundamentais que ajudam a estruturar e formatar o conteúdo na página da web. Cada tag HTML é representada por um nome envolto em colchetes angulares <>.

Exemplos de tags e suas funcionalidades

Estrutura Básica:

  • <html>: Define o início e o fim do documento HTML.
  • <head>: Contém informações sobre o documento, como título, meta tags e links para estilos.
  • <title>: Define o título da página exibido na barra de título do navegador.
  • <body>: Contém o conteúdo principal da página.

Textos:

  • <h1>, ..., <h6>: Define cabeçalhos de diferentes níveis.
  • <p>: Define parágrafos.
  • <strong> ou <b>: Define texto em negrito
  • <em> ou <i>: Define texto em itálico.
  • <u>: Sublinha o texto.

Links e Imagens:

  • <a>: Cria links para outras páginas ou recursos.
  • <img>: Insere imagens.

Listas:

  • <ul>: Define uma lista não ordenada.
  • <ol>: Define uma lista ordenada.
  • <li>: Define itens de lista.

Tabelas:

  • <table>: Define uma tabela.
  • <tr>: Define uma linha em uma tabela.
  • <th>: Define uma célula de cabeçalho em uma tabela.
  • <td>: Define uma célula de dados em uma tabela.

Formulários:

  • <form>: Define um formulário.
  • <input>: Cria campos de entrada em um formulário.
  • <text>: Cria uma área de texto em um formulário.
  • <button>: Cria botões em um formulário.

Multimídia:

  • <audio>: Incorpora áudio em uma página.
  • <video>: Incorpora vídeo em uma página.

Estilo e Layout:

  • <div>: Define uma divisão ou seção genérica em um documento.
  • <span>: Define uma pequena seção de texto em linha.
