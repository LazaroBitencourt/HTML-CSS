Seletores CSS

Bem-vindo!

No CSS nós possuímos os seletores por ID

  • Seletor por Tipo: Busca elementos por uma tag HTML específica
  • Seletor por ID(#:)Busca elementos através do atributo "id"
  • Seletor por Classe(.):Busca elementos através do atributo "class"
  • Seletor Universal:Seleciona todos elementos do HTML
  • Seletor de Atributo([atrib]):Seleciona elementos que possuem um atributo específico em noss documento
  • Seletor de Atributo([atrib]):Seleciona elementos que possuem um atributo específico em noss documento HTML e conseguimos também buscar com um valor específicco
